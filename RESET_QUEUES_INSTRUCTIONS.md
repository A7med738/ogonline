# إعادة تعيين الطوابير وإضافة بيانات حقيقية - نظام احجزلي

## 🎯 **الهدف:**
صفير جميع الطوابير وإضافة بيانات حقيقية لكل دكتور وعيادة.

## 📋 **ما سيتم تنفيذه:**

### **1. تنظيف البيانات:**
- حذف جميع المواعيد الموجودة
- إعادة تعيين عدد المرضى المنتظرين لجميع العيادات

### **2. تحديث بيانات العيادات:**
- **11 عيادة** بأسماء وأطباء حقيقيين
- **تخصصات متنوعة** (طب عام، أطفال، نساء، قلب، عظام، إلخ)
- **أسعار مختلفة** حسب التخصص (150-320 ج.م)
- **سعات مختلفة** حسب نوع العيادة (12-30 مريض/يوم)

### **3. إضافة مواعيد تجريبية:**
- **5 مواعيد اليوم** (pending) - للاختبار
- **3 مواعيد أمس** (completed) - للتاريخ
- **2 مواعيد قبل أمس** (completed) - للتاريخ

## 🚀 **خطوات التنفيذ:**

### **الخطوة 1: تشغيل السكريبت**
1. اذهب إلى **Supabase Dashboard**
2. اختر **SQL Editor**
3. انسخ محتوى `RESET_QUEUES_AND_ADD_REAL_DATA.sql`
4. اضغط **Run**

### **الخطوة 2: التحقق من النتائج**
بعد تشغيل السكريبت، ستظهر:
- قائمة بالعيادات مع عدد المرضى المنتظرين
- قائمة بالمواعيد اليوم مع أرقام الطابور

### **الخطوة 3: اختبار النظام**
1. اذهب للتطبيق
2. تصفح المراكز الصحية
3. اختر عيادة واعرض العيادات
4. جرب حجز موعد جديد
5. تحقق من صفحة المواعيد
